Projects

Spark

There are many websites to help you research and plan a trip once you've chosen a destination. But how do you choose an appropriate destination from the endless options around the globe? Spark is a travel website designed to inspire users to travel, help them choose an appropriate destination, and provide the best resources for trip planning.

The inspiration page uses Instagram feeds and the Instafeed.js library to display wanderlust-inducing photos with their locations (when provided). Once the user figures out what sparks their desire to travel, the search page lets them search for the perfect destination based on region, price, budget, 'sparks', etc. With destination ideas in mind, the user can browse the best 3rd-party resources for further research and bookings.

  • AngularJS Single Page Application
  • Node.js & Express.js Backend
  • MongoDB database for searching destinations
  • Displays public Instagram feeds
  • Built using Bootstrap & jQuery

Amazon Alexa Skills

Alexa is Amazon's intelligent personal assistant, which is available in the Amazon Echo and several other devices. I have three applications, known as 'skills', that are published in the Amazon Alexa App store. The skills use the Alexa voice-based user interface and Speech Synthesis Markup Language. I really enjoyed learning and thinking about how to design an effective voice interface. Together, these skills received about 2,500 utterances from users in their first 4 months!

Learn International Geography is a short trivia game to practice geography knowledge, while Marine Life provides users with quick, interesting facts about various ocean creatures. International Calling Codes allows users to find countries and their associated dialing codes.

  • 3 skills published by Amazon
  • JavaScript & Node.js
  • Amazon Lambda
  • Amazon's NoSQL DynamoDB database
  • voice-based user interactions

Bloc Jams

Bloc Jams is an interactive music streaming website. Users can select an album, then play, pause, and skip songs. They can jump to any point in the song, or change the volume, using slider buttons. Playback data is collected and displayed in a pie chart. For learning purposes, this project was written in vanilla JS, then refactored using jQuery, and then rebuilt using AngularJS.

  • AngularJS & jQuery
  • Interactive music playback controls
  • Audio file management using Buzz.js
  • Data Visualisation using Angular-nvd3

Bloccit

Bloccit is a Ruby on Rails application inspired by Reddit. Users can create an account, publish posts and comments, use tag labeling, and vote items up or down. Users can also favorite posts in order to receive email updates about them. Bloccit provides user profiles and three levels of user permissions.

  • Ruby on Rails
  • Responsive Design with Bootstrap
  • PostgreSQL database
  • User authentication
